**Одна программа — всё, что нужно для управления Windows.**  
SysMasterUltra объединяет **10+ функций** в едином удобном интерфейсе. Мониторинг, процессы, сеть, шифрование, хуки клавиатуры — всё в одном окне.

---

## 🎯 Зачем это нужно?

| Проблема | Решение SysMasterUltra |
|----------|------------------------|
| Нужно быстро посмотреть загрузку CPU/RAM | Виджеты и график в реальном времени |
| Студенты теряют файлы, менеджеры устали восстанавливать | Автоархиватор + менеджер файлов |
| Нужно просканировать порты или проверить доступность хоста | Встроенные сетевые утилиты (Ping, сканер портов) |
| Преподаватель просит показать 10 лаб в одной программе | Готово — все лабы в одном окне |
| Хочется красивый интерфейс вместо консоли | Собственные контролы: MaterialButton, GradientPanel |

## ✨ Возможности

### 🔹 Процессы 
- Просмотр всех процессов (имя, PID, память, потоки, приоритет)
- Завершение процессов
- Запуск новых процессов с заданным приоритетом
- Демонстрация таймеров и синхронизации (счётчик, пауза, сброс)

### 🔹 Клавиатура 
- Глобальный хук клавиатуры (перехват нажатий в любой программе)
- Логирование всех нажатий в реальном времени
- Блокировка клавиши ESC (вкл/выкл)
- Счётчик нажатий

### 🔹 Мониторинг 
- Графики CPU и RAM в реальном времени
- Виджеты с показателями (CPU, RAM, процессы, потоки)
- Настраиваемая глубина истории (10–200 точек)
- Статус-бар с текущими значениями

### 🔹 Потоки
- Демонстрация трёх подходов к многопоточности:
  - ThreadPool (пул потоков)
  - Async/Await (современный асинхронный подход)
  - Ручное создание потоков
- Логирование выполнения каждой задачи

### 🔹 Рефлексия
- Загрузка любой .NET-сборки (.dll/.exe)
- Просмотр классов, методов, сигнатур
- Вызов методов с параметрами (динамическое создание экземпляров)
- Поддержка статических и экземплярных методов

### 🔹 Сеть
- **Ping** любого хоста (с таймаутом 3 сек, детальный вывод)
- **Сканер портов** (многопоточный, диапазон 1–65535, таймаут 500 мс)
- Отображение открытых портов в реальном времени

### 🔹 Безопасность
- **Шифрование/дешифрование текста (AES)**
- **Криптостойкий генератор паролей** (RNGCryptoServiceProvider)
- Настройка длины пароля и набора символов
- Генерация нескольких паролей сразу

### 🔹 Утилиты
- **Очистка временных файлов** (с подсчётом размера)
- **Скриншот рабочего стола** (с сохранением на рабочий стол)
- **Детальная информация о системе** (ОС, память, диски, сеть)
- **Менеджер автозагрузки** (просмотр файлов в автозагрузке)
- **Диспетчер служб** (информация и команды)
- **Просмотр событий** (журналы Windows)
- **Редактор реестра** (информация и безопасные команды)

### 🎮 Мини-игры
- **«Угадай число»** (компьютер загадал, ты угадываешь)
- **«Тест реакции»** (нажми на зелёную кнопку как можно быстрее)

## 🏗 Архитектура

Проект построен по модульному принципу с разделением на слои:
```
SysMasterUltra/
├── Forms/ # Формы (главное окно)
│ ├── MainForm.cs # Основная форма (1000+ строк кода)
│ └── MainForm.Designer.cs
├── Controls/ # Собственные контролы
│ ├── MaterialButton.cs # Кнопка с анимацией свечения
│ └── GradientPanel.cs # Панель с градиентным фоном
├── Services/ # Сервисы (бизнес-логика)
│ ├── KeyboardHookService.cs # Глобальный хук клавиатуры
│ ├── PerformanceMonitor.cs # Мониторинг CPU/RAM
│ └── Logger.cs # Логирование
├── Properties/ # Настройки проекта
├── Program.cs # Точка входа
└── App.config # Конфигурация .NET
```

**Ключевые архитектурные решения:**
- **Модульность** — каждый функциональный блок вынесен в отдельный сервис
- **Собственные контролы** — для кастомизации интерфейса
- **Событийно-ориентированная архитектура** — для хуков и мониторинга
- **Многопоточность** — для сканера портов и демонстрации потоков
- **P/Invoke** — для низкоуровневого доступа к WinAPI
